### Getting Started
```
# Clone the repo
git clone https://github.com/katwal-dipak/react-native-starter-kit.git

# Navigate to project folder and Install dependencies
npm install

# Install Pods
cd ios && pod instal
```
### Run the React Native App
```
npx react-native run-android
npx react-native run-ios
```
